🎤 **AudioScribe**

AudioScribe — проект настольного приложения для автоматической транскрибации аудиофайлов в текст с использованием технологий распознавания речи (ASR).

Проект находится на стадии разработки.

⸻

📌 **Описание проекта**

AudioScribe предназначен для преобразования аудиозаписей в текст в автоматическом режиме. Система будет работать локально на устройстве пользователя и не требовать постоянного подключения к интернету.

Приложение планируется как удобный инструмент для расшифровки лекций, интервью, встреч, подкастов и других аудиоматериалов.

🖥️ **Архитектура системы**

Приложение будет состоять из двух основных компонентов:

🖼️ Пользовательский интерфейс

Отвечает за:
	•	выбор файла
	•	запуск обработки
	•	отображение статуса
	•	вывод результата

⸻

⚙️ Модуль обработки аудио

Отвечает за:
	•	загрузку модели распознавания речи
	•	обработку аудиофайла
	•	сегментацию длинных записей
	•	распознавание текста
	•	формирование итогового результата

⸻

🔁 Схема работы

User → GUI → Audio Processing Module → ASR Model → Text → Save Result


⸻

⚙️ **Требования к системе**

Программные требования
	•	Windows / macOS / Linux
	•	Python 3.10+
	•	ffmpeg (для обработки аудио)
	•	возможность установки зависимостей Python

Интернет потребуется только для первоначальной загрузки моделей.

⸻

Аппаратные требования

Минимальные:
	•	CPU: 2 ядра

	•	RAM: 8–16 GB
