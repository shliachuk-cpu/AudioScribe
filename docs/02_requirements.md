# Требования (MVP)

## MUST (обязательные)

### R1 — Выбор аудиофайла
Пользователь может выбрать локальный файл.

### R2 — Поддержка форматов
Поддерживаются MP3, WAV, FLAC.

### R3 — Транскрибация
Речь преобразуется в текст.

### R4 — Отображение результата
Текст показывается в интерфейсе.

### R5 — Сохранение результата
Текст можно сохранить в файл.

### R6 — Индикация прогресса
Отображается статус обработки.

## NICE (желательные)
- Поддержка длинных файлов
- Отображение CPU/GPU
- Повторная обработка
- Форматирование текста
