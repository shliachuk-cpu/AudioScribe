# UX и сценарии

## Главное окно

Элементы:
- Поле выбранного файла
- Кнопка выбора
- Кнопка запуска
- Кнопка сохранения
- Строка статуса
- Текстовое поле результата

## Состояния UI
- Empty
- Loading
- Success
- Error

## Сценарии

### Транскрибация
1. Выбор файла
2. Запуск
3. Получение текста

### Сохранение
1. Выполнить транскрибацию
2. Нажать «Сохранить»
3. Выбрать путь

### Ошибка
1. Запуск обработки
2. Возникновение проблемы
3. Показ сообщения

## Типовые ошибки
- Файл не выбран
- Неподдерживаемый формат
- Ошибка обработки
- Недостаточно ресурсов
- Нет ffmpeg
