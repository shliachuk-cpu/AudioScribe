# Нефункциональные требования

## Производительность
- Интерфейс не зависает
- Обработка в фоне

## Надёжность
- Приложение не падает при ошибках
- Понятные сообщения

## Хранение данных
- Нет автоматического хранения
- Сохранение по запросу

## Ограничения платформы
- Desktop-ОС
- Требуется ffmpeg
- Достаточный объём RAM

## Безопасность
- Обработка локально
- Нет передачи данных в интернет
