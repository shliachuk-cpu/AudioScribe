# Описание проекта и границы

## Название проекта
Desktop Audio Transcriber (Parakeet-based)

## Что делает приложение
Приложение предназначено для автоматической транскрибации аудиофайлов в текст с использованием нейросетевой модели распознавания речи. Пользователь загружает аудиофайл, после чего приложение обрабатывает его и отображает распознанный текст с возможностью сохранения в файл.

## Пользователь
- Студенты, журналисты, аналитики, разработчики
- Люди, которым нужно преобразовывать аудио в текст локально
- Пользователи, заботящиеся о приватности

## Функции (MVP)
1. Выбор локального аудиофайла
2. Автоматическая транскрибация речи в текст
3. Отображение результата
4. Сохранение текста в файл
5. Индикация процесса обработки

## Что НЕ делаем
- Онлайн-транскрибацию через облачные сервисы
- Редактирование аудио
- Распознавание говорящих
- Перевод текста
- Потоковую обработку

## Платформа / стек (предварительно)
- Desktop-приложение
- Windows / Linux / macOS
- Python + Tkinter
- PyTorch + NeMo
